A collection of tools for indie game developers and creators, designed to simplify common tasks.

Particle Simulator

A particle simulator using interactions between up to 4 types of particles. Can simulate how attractions and distractions make unique anomalies.

wikiBlocks

Simple yet powerful, file-based wiki that allows you to create your own personal knowledge base using markdown files. It's designed to be easy to set up, edit, and customize.

Sprite Builder

Sprite Builder simplifies the process of creating 2D sprites from 3D models, enabling seamless integration of 3D assets into 2D game environments and applications.

PixelBlocks

Easy to use Image to Pixel art generator with filters/ art canvas and many more features for seamless retro pixel-art creations. Lightweight and powerful tool for pixel-art game devs.

Pixel Texture Gen

procedural tool designed to create pixel-perfect textures for pixel artists and game developers. With 120 built-in presets and customization options, you can generate seamless pixel textures effortlessly.

MarkdownBlocks

A versatile markdown editor with a rich toolbar for formatting, import/export functionality, and theme customization. It allows users to create and edit markdown documents with a live preview.

Textorium

A minimalist markdown reader that supports theme and font toggling along with a dynamically generated table of contents for easy navigation within the document.

Snapshot!

Tool to take window screenshots - allows users to generate visually appealing windowed screenshots of images with customizable window titles and other options.

Snapshot <\code>

Tool to take code screenshots. This tool allows users to generate visually appealing screenshots of code snippets with customizable window titles and language selection for syntax highlighting.

ColorLab

simple and effective one-click color harmonies generator and exporter. ColorLab also features an advanced version to fine-tune color palettes for perfection.
